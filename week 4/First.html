Get to Know HTML

When computer developers are making websites, they code in HTML to create to the webpage. Within HTML, there are certain tags that allow these developers to add certain design elements to webpages. For example, there are tags for headings of the website, paragraphs, lists, etc.

Heading Tags

In Html there are six different tags for headings. They are labeled 1-6 with 1 being the biggest and 6 being the smallest. Below is an example of how the heading and tags will look:

  • < h1 > < /h1 >

    H1 heading

  • < h2 > < /h2 >

    H2 heading

  • < h3 > < /h3 >

    H3 heading

  • < h4 > < /h4 >

    H4 heading

  • < h5 > < /h5 >
    H5 heading
  • < h6 > < /h6 >
    H6 heading

Paragraph Tag

In html there is only one tag for writing paragraphs. Below is an example of what it looks like:

  • < p > < /p >

    This is a paragraph.

Formatting Tags

In HTML, there are multiple tags for formatting words. You can make them bold, italicized, etc. Below are the tags and how they are used:

  • < b > < /b > - Bold text
  • < strong > < /strong > - Important text
  • < i > < /i > - Italic text
  • < em > < /em > - Emphasized text
  • < mark > < /mark > - Marked text
  • < small > < /small > - Smaller text
  • < del > < /del > - Deleted text
  • < ins > < /ins > - Inserted text
  • < sub > < /sub > - Subscript text
  • < sup > < /sup > - Superscript text
